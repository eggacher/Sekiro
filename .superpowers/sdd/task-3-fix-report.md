## 权限矩阵对应关系

根据设计文档规格，修复后的权限分配遵循：

### 菜单 ID 解释
- `1` = 工作台（全部角色可见）
- `2` = 系统管理（目录）
  - `21` = 用户管理 | `211-213` = 新增/编辑/删除按钮
  - `22` = 角色管理
  - `23` = 菜单管理
  - `24` = 部门管理
  - `25` = 岗位管理
  - `26` = 数据字典
- `3` = 系统监控（目录）
  - `31` = 在线用户
  - `32` = 登录日志
  - `33` = 操作日志
  - `34` = 服务监控

### 权限分配原则
- **超管 + 管理员**：拥有最高权限，含全部按钮操作
- **财务/运营**：访问系统管理全部菜单 + 按钮权限
- **市场**：系统管理部分菜单 + 监控权限
- **客服**：仅监控菜单 + 日志查看
- **开发**：系统管理简版 + 完整监控菜单
